Unit Converters¶

class radical.utilities.unit_converters.PiUnitConverter¶

Bases: radical.utilities.unit_converters.AbstractConverter

This is a very basic one-to-one units converter based on the units found in PI and typical units of measure that we expect to see regarding drilling activities.

Attributes:

metric_units (list): A list of recognized metric units.

imperial_units (list): A list of recognized imperial units.

available_units (list): Combined list of metric_units and imperial_units.

metric_imperial (dict): Metric-to-imperial conversion dictionary.

imperial_metric (dict): Imperial-to-metric conversion dictionary.

get_converter(unit)¶

Returns the appropriate units converter dictionary to use, based on the unit given as the argument.

Args:
unit (str): An authorized unit of measurement. See this classes attributes.
Returns:
dict : The appropriate converter dictionary (metric-to-imperial or imperial-to-metric)
Exceptions:
TypeError : Raised if the unit argument is not recognized.
get_units_system(unit)¶

Method to determine the appropriate units system, given the ‘unit’ argument. R

Args:
unit (str): The test unit to classify.
Returns:
list: The first item in the list should be a string indicating the measurement system (‘imperial’ or ‘metric’). The second item is itself a list containing the set of recognized units of measurement within the system.
Exceptions:
TypeError : Raised if unit argument is not recognized.
