Background¶

r.a.d.i.c.a.l¶

rig activity detection and classification using inference and/or machine learning

Purpose¶

The purpose of this software package is to detect and identify certain rig operations that are performed over the course of drilling a well. After those operations or activities are characterized, performance indicators based on the statistics around each operation can be built. An example of an activity would be a connection event while drilling. An example of a performance indicator would be average slips-to-slips time for all connection events.

Statistics such as these are useful to a number of people within the drilling hierarchy.

The activities of interest are…

  • connections
  • reaming
  • cleaning
  • tripping in
  • tripping out
  • drilling - sliding
  • drilling - rotating
  • surveying
